# Ninja-canvas
忍者画板：一款兼容 PC 以及 移动端 的画板，麻雀虽小，五脏俱全～  

敬开发者：  
源代码注释丰富，适合初学 Canvas 以及 Vue 的同学，数据通信采用 EventBus，代码编写条理清晰，注重函数复用，欢迎学习～
